TecBurguer
Sistema web para gerenciamento completo de uma hamburgueria: loja para clientes, controle de estoque, relatórios financeiros e RH para administradores, painel de pedidos para cozinha e entregador.

Visão geral
Funcionalidades principais

Loja para usuários: catálogo, carrinho e checkout.

Admin: controle de estoque, relatórios de lucros e gestão de RH.

Cozinha: painel em tempo real com pedidos a preparar.

Entregador: painel com pedidos prontos e atualização de status.

Deploy: aplicação publicada no Railway em https://tecburguer-production.up.railway.app.

Stack

Componente	Tecnologia
Back end	C# (.NET) MVC
Front end	HTML; CSS; JavaScript
Banco	Supabase (Postgres via Npgsql)
Deploy	Railway
API externa	BuscaCEP (serviço de CEP usado para autocompletar endereços)
